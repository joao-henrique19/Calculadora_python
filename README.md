# Calculadora em Python

Calculadora feita em Python com interface de terminal.

## Funções

- Soma
- Subtração
- Multiplicação
- Divisão
- Potência
- Raiz quadrada
- Seno
- Cosseno
- Tangente
- Logaritmo
- Fatorial

## Tecnologias

- Python
- Biblioteca math
- ANSI colors (terminal)
